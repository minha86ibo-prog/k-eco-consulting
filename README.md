# K-ECO CONSULTING

> Sustainable Energy Platform
> 폐자원을 에너지로, 에너지를 미래 가치로.

에너지 기술(Ecoflame)과 금융을 연결하여 RE100, ESG, 탄소중립 솔루션을 제공하는 기업 홈페이지입니다.

---

## Overview

K-ECO CONSULTING의 브랜드 아이덴티티를 기반으로 제작한 반응형 웹사이트입니다.

기술 → 금융 → 고객으로 이어지는 사업 구조를 시각적으로 표현하고, RE100 솔루션 및 에너지 사업 정보를 효과적으로 전달하는 데 중점을 두었습니다.

---
